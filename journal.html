📋 조교일지 작성기

시대학원 — 입력 → 카톡 복사 + 시트 자동 제출
미연결
v2.0

🎯 이 도구는?

조교일지를 구조화된 양식에 빠르게 입력하면, 카카오톡에 바로 복사할 수 있는 정리된 텍스트가 자동 생성됩니다.
동시에 Google 스프레드시트에 자동 제출되어, 원장님이 한 화면에서 전체 조교 일지를 날짜별로 조회·관리할 수 있습니다.

📝 작성 순서

1
기본 정보 — 날짜(자동), 조교 이름, 업무 시작/종료 시간 입력. 근무시간은 자동 계산.
2
수업별 업무 — 학년 선택 → 업무유형 → 범위 → 학생별 이름·상태·상세를 입력. 수업이 여러 개면 [+ 수업 추가].
3
학생 지각/출결 — 지각 학생 이름, 분, 사유를 기록.
4
기타 메모 — 특이사항 직접 입력 또는 태그 버튼 클릭.
5
하단의 [📋 카톡 + 시트 제출] 버튼 클릭 → 카톡에 붙여넣기(Ctrl+V) + 스프레드시트 자동 기록!

💡 유용한 팁

🎓 데모 버튼 — 상단 오른쪽 [🎓 데모] 버튼을 누르면 예시 데이터가 자동 입력됩니다. 처음 사용할 때 흐름을 익히기 좋습니다!
⚙️ 최초 1회 설정 필요 — [설정] 탭에서 Google 스프레드시트 연결 설정을 해야 시트 제출이 작동합니다. 가이드를 따라 5분이면 완료!
📋 카톡만 복사도 가능 — 시트 연결 전에도 카톡 복사 기능은 정상 작동합니다.
📱 모바일 지원 — 스마트폰에서도 사용 가능합니다.
